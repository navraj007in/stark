pub fn parse_with_limits(input: &str, limits: &CsvLimits) -> Result<Vec<Vec<String>>, CsvError>

Parse UTF-8 CSV text: comma-delimited, double-quote-quoted, doubled-quote-escaped, LF or CRLF input record separators. Returns decoded records in source order; quote syntax is removed and doubled quotes become one quote. Malformed input returns a typed [CsvError], never a trap.
