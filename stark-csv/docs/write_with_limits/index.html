pub fn write_with_limits(records: &Vec<Vec<String>>, limits: &CsvLimits) -> Result<String, CsvError>

Serialize records to canonical CSV text: CRLF between records (none trailing), minimal deterministic quoting (only when a field contains a comma, quote, CR or LF), doubled quotes, exact UTF-8 preservation. A record with zero fields writes as an empty record (indistinguishable from a one-empty-field record; re-parsing yields [""] for it -- CSV cannot represent the difference).
