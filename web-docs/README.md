# ⚠️ Archived Web Documentation (Pre-Pivot Design)

These HTML pages document the original, pre-pivot STARK design (AI-native
types, hybrid GC, actor concurrency, cloud primitives, lowercase `i32`/`f32`
type names). They are **superseded** by the normative Core v1 specification in
[`STARKLANG/docs/spec/`](../STARKLANG/docs/spec/) and are retained for
historical reference only. See
[`STARKLANG/docs/archive/README.md`](../STARKLANG/docs/archive/README.md) for
the list of known conflicts with the current spec.
